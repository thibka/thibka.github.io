Options

label (string)

Label displayed at the top of the panel.

const gui = new GUI({ label: 'Basics' });

draggable (boolean)

Determines whether the panel can be moved manually.
Default is false. 

const gui = new GUI({ draggable: true });

container (string|HTMLElement)

Parent element of the panel. Can be a string selector or a DOM element.
Default is document.body.

const gui = new GUI({ container: '.container' });

position (string)

GUI instances can be positioned in any corner of the screen / container.
When multiple instances share the same position (like GUI 1 and GUI 2 in the example below), they are stacked next to each other.
Possible values are top left, top right, bottom left, bottom right.
Default is top right. 

const gui = new GUI({ position: 'top left' });

width (number)

Determines the width of the panel.
Default is 290.

const gui = new GUI({ width: 200 });

closed (boolean)

Just like folders, GUI panels can be closed by default by setting the closed option to true.
Default is false. 

const gui = new GUI({ closed: true });

maxHeight (number)

Determines to define the maximum height of a panel beyond which scrolling is necessary.
Default is the smallest value between the height of the window and the height of the container. 

const gui = new GUI({ maxHeight: 400 });

color (string)

Can be applied to GUI panels and folders to customize the background color.

const gui = new GUI({ color: '#ff0000' });

opacity (number)

Sets the default transparency of the panel. When hovered, the panel becomes fully opaque (opacity: 1).
Default is 1. 

const gui = new GUI({ opacity: 0.5 });

onUpdate (function)

Callback function, triggered each time the GUI is updated.

const gui = new GUI({ onUpdate: () => console.log('updated') });

Methods

.button( parameters )

Creates a clickable button. Pass an optional options object to customize its appearance and behavior.

gui.button( options ).onClick( callback )

Returns the button instance, allowing you to chain .onClick(callback).

onClick(callback) is called whenever the button is clicked.

Options

Parameter Type Description
label string Text displayed inside the button.
color string Accent color of the button.
hoverColor string Accent color when the button is hovered. Defaults to color if provided.
tooltip string|boolean Tooltip displayed on hover. If true, the label is used as the tooltip. 
import GUI from 'perfect-gui';

const gui = new GUI();

gui.button({ 
    label: 'Button', 
    color: '#bb33bb', 
    hoverColor: '#cc3333' 
}).onClick(changeColor);

.slider()

Creates a slider bound to a numeric property. The slider automatically keeps the property and the UI in sync.

Pass an object, the name of one of its properties, and an optional options object to bind the slider to that property. The slider automatically keeps the object property and the UI in sync. Updating the slider changes the property, and assigning a new value to the property updates the slider.

gui.slider( object, property, options ).onChange( callback )

Returns the component instance, allowing you to chain the .onChange(callback).

The callback receives the updated numeric value.

Options

Parameter Type Description
label string Displayed label. Defaults to the property name.
min number Default is 0.
max number Default is 1.
step number Amount by which the value changes.
Default is calculated so that there are 100 steps between min and max.
tooltip string|boolean Descriptive text showing up on hover. If set to true, its value will be the same as the label. 
import GUI from 'perfect-gui';

const position = { x: 0 };

const gui = new GUI();

gui.slider(position, 'x', { 
    label: 'Slider', 
    min: -30, 
    max: 30 
}).onChange((value) => {
    element.style.transform = `translateX(${value}px)`;
});

.toggle()

Creates a toggle bound to a boolean property. The toggle automatically keeps the property and the UI in sync.

Pass an object, the name of one of its properties, and an optional options object to bind the toggle to that property. The toggle automatically keeps the object property and the UI in sync. Toggling the control changes the property, and assigning a new value to the property updates the toggle.

gui.toggle( object, property, options ).onChange( callback )

The bound property should contain a boolean value.

Returns the component instance, allowing you to chain the .onChange(callback). The callback receives the updated boolean value.

Options

Parameter Type Description
label string Displayed label. Defaults to the property name.
tooltip string|boolean Descriptive text showing up on hover. If set to true, its value will be the same as the label. 
import GUI from 'perfect-gui';

const settings = { isRound: false };

const gui = new GUI();

gui.toggle(settings, 'isRound', {
    label: 'Rounded corners',
}).onChange((value) => {
    element.classList.toggle('round', value);
});

.list()

Creates a dropdown list bound to a property. The list automatically keeps the property and the UI in sync.

Pass an object, the name of one of its properties, an array of values, and an optional options object to bind the dropdown list to that property. The list automatically keeps the object property and the UI in sync. Selecting a different option updates the property, and assigning a new value to the property updates the selected option.

gui.list( object, property, values, options ).onChange( callback )

The values parameter accepts either:

  • an array of strings or numbers
  • an array of { label, value } objects, where label is displayed in the dropdown and value is assigned to the bound property

Options

Parameter Type Description
label string Optional.
tooltip string|boolean Descriptive text showing up on hover. If set to true, its value will be the same as the label. 
import GUI from 'perfect-gui';

const gui = new GUI();

// Using an array of strings

const settings = { color: 'white' };

gui.list(settings, 'color', ['red', 'white', 'yellow', 'blue'], {
    label: 'String array',
}).onChange((value) => {
    element.style.backgroundColor = value;
});

// Using an array of objects
// The "label" property is only used to control what's displayed in the option list
// The underlying value of each item is the "value" property

const settings2 = { color: '#993333' };

gui.list(
    settings2,
    'color',
    [
        { label: 'reddish', value: '#993333' },
        { label: 'pinkish', value: '#aa33aa' },
        { label: 'yellowish', value: '#999933' },
        { label: 'blueish', value: '#333399' },
    ],
    {
        label: 'Object array',
    },
).onChange((obj, index) => {
    element.style.backgroundColor = obj.value;
});

.image( parameters )

Acts similar to a button but uses an image instead of text.

Returns the component instance, allowing you to chain the .onClick(callback).

Parameter Type Description
label string Optional. Default is the image file label.
path string Image file path.
selected boolean Default state of the item. Default is false.
selectionBorder boolean Defines if a border is visible around the selected element. Default is true.
width number | string A number type value specifies the width in pixels, while a string value allows for defining the width using various units of measurement, such as '50%' or '5vw'. Default is 33.333%.
height number Defines the height of the element in pixels. Default is 90.
tooltip string|boolean Descriptive text showing up on hover. If set to true, its value will be the same as the label. 
import GUI from 'perfect-gui';

const gui = new GUI();

gui.image('path/to/image1.jpg', {label: 'Image 1'}).onClick(changeBackground);
gui.image('path/to/image2.jpg', {label: 'Image 2', selected: true}).onClick(changeBackground);
gui.image('path/to/image3.jpg', {label: 'Image 3'}).onClick(changeBackground);
gui.image('path/to/image4.jpg', {label: 'Image 4'}).onClick(changeBackground);
gui.image('path/to/image5.jpg', {label: 'Image 5'}).onClick(changeBackground);

function changeBackground(evt) {
    element.style.backgroundImage = `url( ${evt.path} )`;
}

.color()

Creates a color picker bound to a color property. The color picker automatically keeps the property and the UI in sync.

Pass an object, the name of one of its properties, and an optional options object to create a color picker bound to that property. The color picker automatically keeps the object property and the UI in sync. Updating the picker changes the property, and assigning a new value to the property updates the picker.

gui.color( object, property, options ).onChange( callback )

The bound property should contain a hexadecimal color string (for example '#06ff89').

Returns the component instance, allowing you to chain .onChange(callback).

Options

Parameter Type Description
label string Displayed label. Defaults to the property name.
tooltip string|boolean Tooltip displayed on hover. If true, the label is used as the tooltip. 
import GUI from 'perfect-gui';

const settings = {
    color: '#06ff89'
};

const gui = new GUI();

gui.color(settings, 'color', { label: 'Color' }).onChange(value => {
    element.style.backgroundColor = value;
});

.vector2( object, xProp, yProp, options )

Creates a 2D vector control bound to two properties on the same object. The vector control automatically keeps the properties and the UI in sync. Moving the control updates both properties, and assigning new values to the properties updates the control.

gui.vector2( object, xProp, yProp, options ).onChange( callback )

Returns the component instance, allowing you to chain the .onChange(callback).

The callback receives the updated x and y values.

Options

Parameter Type Description
label string Displayed label. Defaults to "xProp / yProp".
min number Default minimum value applied to both axes.
max number Default maximum value applied to both axes.
step number Default increment used for both axes.
x object Overrides configuration for the X axis.
Can contain: min, max, step.
y object Overrides configuration for the Y axis.
Can contain: min, max, step.
tooltip string|boolean Descriptive text shown on hover.
If set to true, the label is used. 

import GUI from 'perfect-gui';
        
const settings = { x: 0, y: 0 };

const gui = new GUI();

gui.vector2(settings, 'x', 'y', { label: 'Position', min: -50, max: 50 }).onChange((x, y) => {
    element.style.transform = `translate(${x}px, ${-y}px)`;
});

.tabs( parameters )

Parameter Type Description
tabs array Required. Array of tab labels.
active number Optional. Index of the initially active tab. Default is 0.

Method Description
.getTab(index) Returns the GUI instance for the tab at the specified index.
.getActiveTab() Returns the index of the currently active tab.
.getTabElement(index) Returns the DOM element for the tab at the specified index.
.setActiveTab(index) Sets the active tab to the specified index. 
import GUI from 'perfect-gui';

const gui = new GUI({label: 'Tabs' });

const tabs = gui.tabs({
    tabs: ['Controls', 'Settings'],
    active: 1
});

const tab1 = tabs.getTab(0);
tab1.button({ label: 'Action Button' });
tab1.slider({ label: 'Value', value: 0.5, min: 0, max: 1 });

const tab2 = tabs.getTab(1);
tab2.toggle({ label: 'Enable Feature', value: true });
tab2.color({ label: 'Color', value: '#ff0000' });
// Tab methods
console.log('Active tab:', tabs.getActiveTab());
console.log('Tab 0 element:', tabs.getTabElement(0));

// Programmatically switch tabs
tabs.setActiveTab(0);

const guis = [];

let gui_1 = new GUI({
    label: 'GUI 1',
});

gui_1.button({ label: 'Create GUI panel' }).onClick(() => {
    guis[guis.length] = new GUI({
        label: 'Created GUI',
        position: 'bottom left',
        width: 150,
        color: 'red'
    });
});

gui_1.button({ label: 'Kill GUI panel' }).onClick(() => {
    const index = guis.length - 1;
    if ( index >= 0 ) {
        // Removes html elements
        guis[index].kill(); 

        // Frees up memory
        guis[index] = null;

        // Removes null value from array
        guis.splice(index, 1);
    }
});
